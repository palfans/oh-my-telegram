# oh-my-telegram 빠른 시작

## 🚀 30초 만에 시작하기

### 1단계: 텔레그램 봇 생성 (15초)

1. Telegram에서 **@BotFather** 검색
2. `/newbot` 보내기
3. 봇 이름: `OhMySisyphus`
4. 사용자명: `oh_my_sisyphus_bot` (영문+숫자+_)
5. 봇 토큰 받기: `1234567890:ABC...`

### 2단계: 설정 (10초)

터미널에서:
```bash
cd /Users/eunoo/projects/oh-my-telegram
./setup.sh
```

봇 토큰을 붙여넣으세요.

### 3단계: 테스트 (5초)

텔레그램에서 새 봇 검색 후:
```
/start
/sisyphus 안녕!
```

## ✅ 완료!

이제 텔레그램으로 sisyphus와 대화하며 코딩 작업을 할 수 있습니다!
