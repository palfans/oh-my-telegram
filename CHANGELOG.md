# CHANGELOG

## [1.0.0] - 2026-01-30

### Added
- Initial release of oh-my-telegram
- Telegram bot interface for oh-my-opencode
- Support for sisyphus, oracle, prometheus, librarian, metis agents
- Session management per Telegram chat
- User whitelist security
- Long message chunking (4000 chars)
- Polling mode support
- JSON and .env configuration
- CLI entry point

### Features
- `/agent` commands to switch agents
- Default agent configuration
- OpenCode CLI bridge
- Real-time execution
- Typing indicators
- Help commands
